# XAlgo

A real-time competitive coding platform where two players are matched against each other to solve algorithmic problems under a time limit.

---

## Features

- 1v1 real-time coding battles with a 20-minute timer
- Live opponent progress tracking via polling
- XP-based ranking and leaderboard
- Forfeit detection with XP penalty

  
---

## Tech Stack:
**React.js** | **Node.js** | **Express.js** | **MongoDB**
