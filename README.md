# 🚀 XAlgo - 1v1 Competitive Coding Arena

A 1v1 competitive coding website where players can challenge each other to live coding battles, solve programming problems in real-time, and compete to see who can solve them first.

---

## 🌟 Key Features

*   **Real-time Coding Battles (1v1):** Challenge other online players and compete in real-time.
*   **Active Player Lobby:** See who is online in a live lobby list and challenge them instantly.
*   **Interactive Code Editor:** Write code in a beautiful text editor (Monaco Editor) that supports C++ styling.
*   **Real-time Progress Tracker:** Track how many test cases your opponent has passed in real-time while you write your solution.
*   **Automatic Matchmaking Queue:** Enter a matchmaking queue to automatically find and play against other online players.


---

## 🛠️ Tech Stack

*   **Frontend:** React, HTML, CSS, JavaScript (TypeScript for type safety)
*   **Backend:** Node.js, Express, Socket.io (for real-time chat/events)
*   **Database:** MongoDB (using Mongoose to store users and coding problems)
